#### Тестовое задание

Должно быть реализовано на Разрешается Meteor.js + Vue.js.
использование любых необходимых npm/meteor пакетов.

Необходимо Приложение создать приложение Time Tracker.
должно поддерживать функции добавления новой задачи, которое
содержит title и description задачи, возможность удаления задачи.

Можно начать отслеживание времени – запускается таймер,
поставить отслеживание времени на паузу – таймер
останавливается, пометить задачу как выполненную задача
перечёркивается и ставится сверху в списке выполненных. Список
выполненных задач идёт снизу, после списка задач доступных к
выполнению.
